Các bước cần làm:

- Dùng npm:
1. Chạy câu lệnh:  npm install

2. Tạo file .env, bằng cách copy/paste nội dung 
file .env.example

3. Cập nhật các tham số của Vnpay tại file .env

3. Để chạy project, tại root: npm run dev
